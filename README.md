# Hockey Kledingregistratie – Uitgebreide Webapp

Deze webapp bevat:

- Registratie van spelers + kledingmaten
- Automatische voorraadafboeking
- Export naar CSV
- Team-filter
- Beheerderslogin
- Voorraadbeheer
- Firebase-ready structuur

Bestanden:
- `index.html` – UI + container voor scripts
- `style.css` – Layout & design
- `app.js` – Registratielogica + tabel + filters
- `voorraad.js` – Voorraadbeheer
- `admin.js` – Beheerderslogin
- `firebase.js` – Firebase (optioneel)

Plaats alle bestanden samen in een Gist of map.  
Open `index.html` in je browser om de app te gebruiken.
